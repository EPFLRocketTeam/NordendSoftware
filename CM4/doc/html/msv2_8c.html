Compute the CRC for a data packet.

Function taken from Maxon Serial V2 specification manual.

Parameters
p_data_arrayPointer to the data array of the packet.
lengthLength of the packet
Note
The p_data_array data must contain two empty bytes [0x00, 0x00] at the end, where the CRC goes.
